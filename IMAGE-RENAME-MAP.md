## STEP 8: Compress ALL images

Before committing, run ALL images through https://tinypng.com
- Upload in batches of 20
- Download compressed versions
- Replace originals with compressed ones
- This typically saves 60-80% file size


## STEP 9: Deploy

```bash
cd Prepkar
git add -A
git commit -m "add all images — jobs, stories, sections, pages, branding"
git push origin main
```

The site will auto-switch from emojis to real images. No code changes needed.
